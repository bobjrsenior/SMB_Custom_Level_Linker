The original GameCubde Rebuilder is done by BSV (bsv798@gmail.com)

This version was created by decompiling version 1.0 and removing the GUI.

Use:

The first command line parameter is the path to the iso.
Every subsequent parameter is the path to a file which matches a file in the iso.

Example:

gcr_v1.0 "Super Monkey Ball (USA).iso" STAGE001.lz st001.gma st001.tpl
